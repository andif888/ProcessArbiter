ProcessArbiter Installation and Setup  

(Un)Installation Instructions

 

Using MSI Installer Package


For easy installation the setup is provided as Mirosoft Installer Package.

  1. Extract the contents of the ZIP file somewhere on your harddisk.
  2. In the extracted files doubleclick setup.exe file.
  3. Follow the installation wizard to complete.

 

 

Uninstall the MSI-Installation Package

  1. Open Control Panel.
  2. Use Add or Remove Programs to uninstall ProcessArbiter.
